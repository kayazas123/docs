Alpine Linux First Steps
========================

-- Enable community and testing repositories:

   $ sudo vi /etc/apk/repositories
   Uncomment wanted repos

-- Update system packages:

   $ sudo apk update

-- Install useful packages:

   - Basic packages:     sudo apk add bash gdb nano lldb
   - Build packages:     sudo apk add clang cmake gcc g++ make ninja
   - Debugging packages: sudo apk add gdb lldb strace musl-dbg openjdk8-dbg
   - man packages:       sudo apk add man man-pages less less-doc
   - SVN packages:       sudo apk add git subversion
   - Dev packages:       sudo apk add gradle openjdk8 zlib-dev
   - GUI fonts:          sudo apk add font-noto

-- Install optional packages:

   - lldb build prereq.: sudo apk add python2-dev ncurses-dev libedit-dev libxml2-dev swig yaml-dev

-- Set bash as default shell:

   - Create .profile in home folder
   - Add "SHELL=/bin/bash"

-- ~/.bashrc convinience updates (append at EOF):

   force_color_prompt=yes
   bind TAB:menu-complete
   alias cd..="cd .."
   alias ll="ls -l"

-- Disable sudo password for sudoers:

   - $ sudo visudo

   - Add the 3rd line below:
     # Allow members of group sudo to execute any command
     %sudo   ALL=(ALL:ALL) ALL
     shahar  ALL=NOPASSWD: ALL

-- Configure git:

   $ git config --global user.name username
   $ git config --global user.email user@domain.com
